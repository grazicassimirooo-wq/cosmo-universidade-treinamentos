# Cosmo - Universidade de Treinamentos 🌌📚

Sempre acreditei que o aprendizado corporativo não precisa ser monótono. Vivenciando na prática o desafio de engajar times em capacitações diárias e em eventos intensos, percebi que o formato faz toda a diferença. Foi pensando nisso que decidi unir minha paixão por Treinamento & Desenvolvimento com a tecnologia! 🚀

Mais do que um simples LMS (Learning Management System), este projeto é o Front-end de uma plataforma focada em **Social Learning** e **Microlearning**. O objetivo? Transformar o treinamento corporativo em uma experiência fluida, gamificada e tão engajadora quanto rolar o feed de uma rede social.

## 🎯 O que foi construído
* **Interface Moderna e Responsiva:** Layout imersivo que se adapta à tela, proporcionando uma experiência de "Dashboard" de alto nível.
* **Modo Escuro (Dark Mode):** Interatividade via JavaScript para alternar entre o tema claro e o tema espacial/futurista escuro.
* **Gamificação:** Sistema visual de pontuação (⭐ pts) e barras de progresso nas "Trilhas em Andamento" para estimular o aprendizado contínuo.
* **Foco em UX/UI:** Elementos desenhados para retenção de atenção, como pílulas de conhecimento (estilo "stories") e um feed de conteúdo rico.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **Tailwind CSS:** Estilização utilitária avançada e design system diretamente no markup.
* **JavaScript (Vanilla):** Lógica para interatividade (como a troca de temas).

## 🚀 Como visualizar o projeto
Basta fazer o download do arquivo `index.html` e dar um duplo clique para abri-lo em qualquer navegador de internet (Chrome, Edge, Firefox, etc.). Não é necessária nenhuma instalação adicional!
